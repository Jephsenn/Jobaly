## üì• Installation

### Step 1: Install the App

1. Download the installer (file should end in `.exe` for Windows or `.dmg` for Mac)
2. Run the installer
3. Follow the installation wizard
4. Launch "Job Search Assistant" from your desktop or Start menu

### Step 2: Install the Chrome Extension

1. Open **Google Chrome**
2. Go to `chrome://extensions/`
3. Turn on **Developer mode** (toggle in top-right corner)
4. Click **Load unpacked**
5. Navigate to where the app is installed, then select the `extension` folder
6. The extension should now appear in your Chrome toolbar

## üöÄ Getting Started

### 1. Upload Your Resume

1. Click **Resumes** in the sidebar
2. Click **Upload Resume**
3. Select your resume file (.docx or .pdf)
4. The app will parse your skills and experience

### 2. Start Job Hunting

#### Option A: Browse Jobs Normally
1. Go to LinkedIn, Indeed, or Glassdoor in Chrome
2. Browse jobs as you normally would
3. When you view a job posting, the extension will automatically capture it
4. You'll see a "‚úì Job captured" notification

#### Option B: Copy Job URLs
1. Copy a job URL from any website
2. The app automatically detects it
3. The job appears in your Dashboard

### 3. Review Matches

1. Open the **Dashboard**
2. See all detected jobs with match scores
3. Click **Details** to see:
   - Full job description
   - Salary information
   - Match score breakdown
   - Required skills

### 4. Track Applications

1. Click **Save** on jobs you want to apply to
2. Mark jobs as **Applied** when you submit
3. Go to **Applications** tab to see your pipeline
4. Add notes about interviews, follow-ups, etc.

## üÜò Troubleshooting

### Extension Not Working
1. Make sure Chrome extension is enabled (`chrome://extensions/`)
2. Refresh the job posting page
3. Check that you're on LinkedIn, Indeed, or Glassdoor

### Jobs Not Appearing
1. Wait a few seconds after viewing a job
2. Check the Dashboard - it might already be there
3. Try copying the job URL instead

### App Won't Start
1. Try restarting your computer
2. Reinstall the app
3. Make sure Windows/Mac isn't blocking it

### Match Scores Not Showing
1. Make sure you've uploaded a resume
2. Refresh the Dashboard
3. Wait a moment - calculations can take a few seconds
